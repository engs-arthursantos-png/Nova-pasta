# Projeto HTML + CSS

Este projeto contém uma página web simples criada com HTML e CSS.

# Estrutura do projeto

- `index.html` → estrutura da página  
- `style.css` → estilização da página  

# Elementos presentes

- Título  
- Parágrafo  
- Lista  
- Link  
- Imagem  
- Botão com efeito hover  

# Como executar

1. Salve os arquivos na mesma pasta  
2. Abra o arquivo `index.html` no navegador  

# Tecnologias usadas

- HTML5  
- CSS3
